# Heart Disease Prediction Using Machine Learning

## Overview

This project predicts the likelihood of heart disease using Machine Learning techniques and patient health information.

## Dataset

Heart Disease Dataset

- Records: 1025
- Features: 14

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Machine Learning Model

Random Forest Classifier

## Results

- Accuracy: 98.53%

## Visualizations

- Target Distribution
- Age Distribution
- Correlation Heatmap
- Confusion Matrix

## Future Scope

- Use larger healthcare datasets
- Compare multiple ML algorithms
- Deploy as a web application
